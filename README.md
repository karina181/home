# home
«Безмолвный пациент» Алекса Михаэлидеса — захватывающий психологический триллер о художнице Алисии, которая убила мужа и замолчала навсегда, и психотерапевте Тео, пытающемся раскрыть тайну её молчания. Книга известна неожиданным финалом, отсылками к греческой мифологии (Алкеста) и глубоким погружением в темы травмы и психоанализа.
