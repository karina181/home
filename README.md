# home
«Безмолвный пациент» Алекса Михаэлидеса — захватывающий психологический триллер о художнице Алисии, которая убила мужа и замолчала навсегда, и психотерапевте Тео, пытающемся раскрыть тайну её молчания. Книга известна неожиданным финалом, отсылками к греческой мифологии (Алкеста) и глубоким погружением в темы травмы и психоанализа.
Описание книги — это краткое изложение ее содержания, сюжета и основной темы, призванное заинтересовать читателя, а также формальные данные (автор, издательство, год), которые помогают идентифицировать издание. Обычно включает аннотацию (на обложке), синопсис (для издателей), а также может включать анализ сюжета, героев, проблематики и личные впечатления, как в случае сочинения по книге.
